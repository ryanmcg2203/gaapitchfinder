# Derived Data

This directory is for dataset files that are generated from the canonical
`gaapitchfinder_data.csv` source file.

Typical flow:

- `gaapitchfinder_data.csv` is the base dataset.
- enrichment or analysis scripts read that file.
- durable generated datasets and coverage reports are written here.
- site assets and charts are generated separately under `site/` and `output/`.

Current files:

- `osm_coverage_report.csv`: OSM pitch coverage report
- `pitch_geometry.csv`: generated by `scripts/enrich_pitch_geometry.py` when run locally
- `club_wikipedia_links*.csv`: review-first Wikipedia/Wikidata link suggestions generated by `scripts/enrich_club_wikipedia.py`

## Wikipedia Link Enrichment

`scripts/enrich_club_wikipedia.py` searches Wikidata for possible Wikipedia
links for each grouped club record. It writes suggestions only; links should be
reviewed before being used on the public site.

Useful commands:

```bash
python3 scripts/enrich_club_wikipedia.py --limit 25
python3 scripts/enrich_club_wikipedia.py --county Leitrim --county Monaghan
python3 scripts/enrich_club_wikipedia.py --club "Portobello GAA"
python3 scripts/enrich_club_wikipedia.py --county Leitrim --workers 2
python3 scripts/enrich_club_wikipedia.py --county Leitrim --county Monaghan --output data/derived/club_wikipedia_links_leitrim_monaghan.csv
python3 scripts/enrich_club_wikipedia.py --min-confidence 80
```

The output includes a `match_status`:

- `suggested`: high-confidence candidate with an English Wikipedia page
- `needs_review`: possible match, but not strong enough to trust automatically
- `no_match`: no useful Wikidata candidate found
- `api_error`: Wikidata request failed and should be retried later
