Dataset

The underlying dataset behind GAA Pitch Finder is freely available on GitHub. It contains nearly 2,000 GAA pitches worldwide, including coordinates, elevation, rainfall data, and more.

View the dataset on GitHub →

What's included

  • Club name and pitch name
  • Latitude and longitude (WGS84)
  • Province, county, country and region
  • Elevation (metres above sea level)
  • Annual rainfall and rain days
  • Google Maps directions link

Coverage

  • Ireland — all 32 counties
  • Great Britain
  • North America (USA and Canada)
  • Rest of Europe
  • Australasia
  • Asia
  • Middle East
  • South America

Usage

The data is published openly for research, education, and community use. If you use it in your work, a mention would be appreciated. For enquiries contact gaapitchfinder@gmail.com.
